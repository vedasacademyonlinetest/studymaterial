Static Friction (fₛ)స్థిర ఘర్షణ
Opposes impending motion. Can vary from 0 to f_max.
f_max = μₛN (limiting friction). Body doesn't move if F ≤ f_max.చలనానికి ముందు నిరోధిస్తుంది. f_max = μₛN. F ≤ f_max అయితే చలనం లేదు.
Kinetic Friction (fₖ)గతిక ఘర్షణ
Acts when body is sliding. fₖ = μₖN. Always < static friction. Constant regardless of speed.జారుతున్నప్పుడు పని చేస్తుంది. fₖ = μₖN. స్థిర ఘర్షణ కంటే తక్కువ. వేగంతో మారదు.
Coefficient of Friction (μ)ఘర్షణ గుణకం (μ)
μ = f/N. Depends on surfaces, not area or speed!
μₛ > μₖ always. Ice: μ≈0.05. Rubber: μ≈0.8μ = f/N. ఉపరితలాలపై ఆధారపడుతుంది, వేగంపై కాదు!
μₛ > μₖ ఎల్లప్పుడూ.
Rolling Frictionదొర్లడ ఘర్షణ
Least among all types! That's why wheels are round. Rolling friction << Kinetic friction < Static friction.అన్ని రకాల్లో అత్యల్పం! అందుకే చక్రాలు గుండ్రంగా ఉంటాయి. దొర్లడ << గతిక < స్థిర

⚡ Tips & Tricks

⚡ Friction Tricks | ఘర్షణ గుర్తులు

→Friction always OPPOSES relative motion (not absolute motion!)ఘర్షణ సాపేక్ష చలనాన్ని అడ్డుకుంటుంది
→Static friction is self-adjusting: 0 ≤ fₛ ≤ μₛNస్థిర ఘర్షణ స్వయంగా సర్దుకుంటుంది: 0 ≤ fₛ ≤ μₛN
→Area of contact does NOT affect friction! (Only N and μ matter)స్పర్శ వైశాల్యం ఘర్షణను మార్చదు! (N మరియు μ మాత్రమే)
→On INCLINE: N = mgcosθ (not mg!). So f = μmgcosθవాలుపై: N = mgcosθ. కాబట్టి f = μmgcosθ

Steps for FBDFBD చేయడానికి దశలు

Elevator Problemsలిఫ్ట్ సమస్యలు
• Going UP with acc a: N = m(g+a) [heavy!]
• Going DOWN with acc a: N = m(g−a) [light!]
• Free fall (a=g down): N = 0 [weightless!]• పైకి త్వరణం a: N = m(g+a) [భారంగా!]
• కిందికి త్వరణం a: N = m(g−a) [తేలికగా!]
